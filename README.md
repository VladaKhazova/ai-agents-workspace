# ai-agents-workspace

Приложение, которое показывает, как несколько AI-агентов могут поэтапно обработать одну задачу.

## Описание проекта

Пользователь вводит задачу в поле ввода. 
Приложение показывает простой сценарий работы нескольких агентов:

- первый агент принимает задачу
- второй агент делит её на шаги
- третий агент обрабатывает шаги
- четвёртый агент собирает финальный результат
- 
Здесь используется mock api, имитируем получение данных.

## На чём сделан проект

Проект сделан на:

- React
- TypeScript
- Vite
- CSS(BEM)

---

## Как запускается проект

Проект запускается через Vite, запускаем run dev, переходим на http://localhost:5173/.
